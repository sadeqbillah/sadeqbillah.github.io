Synthetic Image Generator for Korean License Plate Detection

 
  • C++ CLI WinForms
  • OpenCV
  • JSON
 
Description
  • There are more than 16 types of License Plates used in Korean Vehicles
  • Traditional License Plate Detection Algorithms cannot operate on different types of plates simultaneously and Training Data is scarce and costly to train deep networks
  • We proposed and developed synthetic license plate image generation based on handcrafted templates for different plates
  • We verified the pipeline by training tensorflow based object classification network and comparing with market solutions/APIs using the same set of Sample Videos
  • Applied filters and transformations helped us to get better detection rate than any solution in the market by removing the dependency to specific angle of view, weather or lighting condition

Plate Type and Range

  • We can generate only one type of plate at the same time. We support all types of License Plates co-existing on the roads.
  • Initial digits decides the range of the first 2/3 digits of the license plates

Embedded Characters

  • Korean Characters to be embedded within the numbers. We can select all or ignore some of them as some characters are specific to distinct vehicle types.
  • Military plates' templates are not released in commercial use version of the software for security measures.

End Digits

  • Last digits of the plates. Choosing 0001 to 9999 will make all the plates from 1 to 9999.

Rotation and Filters

  • User can choose any rotation angles to expand the angle of detection.
  • Several filters can be used to increase effectiveness in different weather and lighting conditions.

Objective Quality Assessment Method for Stitched Images

 
  • C++ CLI WinForms
  • OpenCV
  • Dulaney Triangulation
  • Affine Transformation
  • PSNR & Geometric Distortion
 
Description
  • Despite many proposed algorithms, the no objective quality evaluation method is developed, so the comparison of algorithms are performed only in subjective way.
  • We proposed a ‘Delaunay-triangulation based objective assessment method’ for evaluating the geometric and photometric distortions of stitched or warped images.
  • The reference and target images are segmented by Delaunay-triangulation based on matched points between two images, the average Euclidian distance is used for geometric distortion measure, and the average or histogram of PSNR for photometric measure.
  • Link to Video Demonstration | Publication

Input Selection

  1. Select Center Image.
  2. Select Cropped Panorama Image created with Left and Right images only.

Filters

Different Filters to Minimize Feature Points in each block to expand the size of Dulaney Triangles

Display

Live display based on the chosen filters and inputs. It shows the locations of the Fetaure Points in both reference and candidate image in selected colors. Also shows a blended preview of the input images, so the user can visualize the difference in the images on a high resolution display.
